## What Should NOT Be Modified

### ⛔ **The Morphystic Doctrinal Core:**

The 22 Major Arcana interpretations in `arcanaManifest` represent a specific Morphystic cosmology and should **not** be altered in the main branch:

- Card names and themes
- Philosophical descriptions (the `text` field)
- The conceptual framework (Forma Nihil, Prison Loop, Cainite Protocol, etc.)

**Why:** These interpretations are derived from *The Black Book of Morphysm* and represent a coherent philosophical system. Changing them would break the doctrinal integrity of the tool.

## How to Contribute

### For Technical Improvements (merge-eligible):

1. **Fork** the repository
2. **Create a branch** for your feature: `git checkout -b feature/your-feature-name`
3. **Make your changes** (code quality, performance, features)
4. **Test thoroughly**
5. **Submit a pull request** with:
   - Clear description of changes
   - Why the change improves the project
   - Any breaking changes noted

### For Alternative Interpretations (fork-only):

If you want to use different tarot systems, numerological methods, or philosophical frameworks:

1. **Fork** the repository
2. **Modify freely** in your fork
3. **Credit the original** SIGIL.FORGE project
4. **Publish independently** as a derivative work (MIT License allows this)

Examples:
- `sigil-forge-thoth` (Crowley's Thoth Tarot)
- `sigil-forge-hermetic` (Golden Dawn system)
- `sigil-forge-chaos` (pure chaos magic, no fixed meanings)
