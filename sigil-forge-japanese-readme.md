# SIGIL.FORGE

****オースティン・オスマン・スペアのシジル技法 --- デジタル実装 v1.0****

「モーフィズムは、人間を尺度とすることの拒絶から生じる。それは牢獄を歪め、腐食させ、その亀裂を探り、システムを崩壊へと押しやることを目指す。」\
--- 『モーフィズムの黒の書』

## 

## I. 概要

****SIGIL.FORGE****
は、オースティン・オスマン・スペアのシジル化技法をウェブ上で実装したツールであり、さらにモーフィスティック哲学に着想を得たタロット型リフレクション・システムを組み合わせたものです。

このツールは、言語的な意図をその核心となる象徴的形態へと還元し、儀式的使用のための視覚的シジルを生成します。

## 

## II. プロセス

### 1. 意図の還元 (\[ EXECUTE.REDUCTION \])

あなたの意図の文を入力してください。\
システムはスペースと重複する文字を削除し、スペアの原典的手法に従って、欲望をその本質的な象徴要素へと蒸留します。

### 2. シジル生成 (\[ FORGE SIGIL \])

還元された文字はキャンバス上のランダムな座標にマッピングされ、唯一無二の幾何学的形態を生成します。

ビジュアル出力は、黒背景に対する高コントラストの赤（#c70c0c）を使用し、鋭く記憶に残るイメージを作り出します。

****伝統的なチャージ方法：****

- シジルをダウンロードして印刷し、燃やして意図を解放する

- 見た直後にデジタルファイルを削除し、意識的な忘却を強制する

- スペアの教義に従い、シジルは集中した注意によってチャージされ、その後、潜在意識で作用するために意識から追放されなければならない

### 3. レゾナンス分析 (\[ ANALYZE.RESONANCE \])

ユニークな文字は数値（A=1, B=2 \... Z=26）に変換され、合計されます。\
その数値は ****22でのモジュロ演算****
によって大アルカナのいずれかに対応づけられます。

各カードはモーフィスティックな視点から再解釈されており、占いではなく、心理的パターンの診断的リーディングを提供します。

- ****ミラー構成****\
  X軸対称により、ロールシャッハ・テストのような心理的投影を喚起します
- ****VHSエフェクト****\
  0.66秒周期のフリッカーとスキャンライン・フィルターが視覚的不安定性を生み出します
- ****グリッチアニメーション****\
  CSSトランスフォームにより、シジルの顕現時に認知的攪乱をシミュレートします

## V. セットアップ

### 前提条件

- Node.js（v14以上）
- npm または yarn

### 

### インストール

1.  リポジトリをクローンします：

git clone https://github.com/yourusername/sigil-forge.git

cd sigil-forge

2.  依存関係をインストール：

npm install

3.  音声ファイルを /public/sounds/ に配置：

- static-void-hum.mp3
- tesla-coil-crackle.mp3
- black-lodge-hum.mp3
- その他（完全なリストは App.jsx を参照）

4.  開発サーバーを起動：

npm start

5.  ブラウザで以下を開く：

[http://localhost:3000](http://localhost:3000/)
